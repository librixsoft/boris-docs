# Installation

This guide will help you install Boris AI on your system.

## System Requirements

- **Java**: 21 or higher (OpenJDK or Oracle JDK)
- **Operating System**: Windows 10+, macOS 10.14+, Ubuntu 18.04+
- **RAM Memory**: Minimum 4GB, recommended 8GB
- **Disk Space**: 1GB of free space

## Installation

1. Download the Boris AI application.
2. Run the executable and follow the instructions on the screen.

## Installation Verification

After installation, verify that Boris AI runs correctly by starting the application.

## Common Issues

### Error: "Java not found"

Make sure you have Java 21+ installed and in your system PATH:
```bash
java -version
javac -version
```
